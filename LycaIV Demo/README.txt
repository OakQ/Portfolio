Lyca IV Test Demo
	Testing the ship's flight controls using Character Controller and box colliders to constrain the ships movements. There are a total of 6 invisible walls surrounding the player so that they can only move within a certain distance. Camera moves with the player. Controls are WASD or Arrow Keys
